A Weather Forecast Application is a real-time web application developed using HTML, CSS, and JavaScript that allows users to check the current weather conditions of any city. It uses a public Weather API to fetch live weather data such as temperature, humidity, wind speed, and weather conditions.

Key Features:

Search weather information by entering a city name.
Display real-time weather details like temperature, humidity, wind speed, and climate status.
Fetch live data using a public Weather API through JavaScript Fetch API.
Provide a simple, responsive, and user-friendly interface.

Technologies Used: HTML, CSS, JavaScript, REST API (Weather API)

Project Outcome:
The application demonstrates API integration, real-time data handling, and dynamic content updating using JavaScript.
